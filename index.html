Faceclaim Directory

Info : Ce faceclaim directory a été créé sr la base de l'annexe d'aide sur les origines. Le faceclaim directory permet de trouver plus rapidement les personnalités ayant les origines que vous recherchez. Ce faceclaim directory est voué à évoluer et à être mis à jour régulièrement. Pour le moment, il n'est qu'à l'état de prototype. Si vous avez des questions, n'hésitez pas à poster dans le sujet : Questions & Suggestions. Merci de votre bienveillance et vos retours.

Disclaimers : Le faceclaim directory a été créé par Elywenn Lubëlior pour le forum Whispering Pines afin d'aider les membres à trouver les faceclaims en fonction des origines. Chaque icon possède son propre crédit, certaines proviennent de graphistes de tumblr, merci pour leur travail.
Les recherches ont été effectuées à l'aide du site ethnicelebs ainsi que des recherches personnelles.
